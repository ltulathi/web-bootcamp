Install Node

In powershell Ctrl-backtick:
npm init
npm install lite-server (this installs node-module)

optional: go into package.json and add start, this creates a shortcut to use npm start as run instead of npx lite-server
npm start (or npx lite-server)

Git Command Lines

git init (initialize repository)
go to github to create the repository
copy and paste the push code from the github site
